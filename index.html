Containers

Operating system virtualization that allow you to run an application and its dependencies in resource-isolated processes.

Docker

Docker is excellent for microservices, as it isolates containers to one process or service.

This intentional containerization of single services or processes makes it very simple to manage and update these services.

Docker Containers

  • Kernel namespaces
